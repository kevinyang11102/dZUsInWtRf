# 学生成绩管理系统 python1156

## 项目概述

学生成绩管理系统是一款软件平台，旨在高效管理学生成绩及相关信息，通过模块化的功能设计，实现信息的全面整合与管理。

## 技术栈

- 后端：Django 2.0, pymysql, click, django-threadlocals, requests, xlrd 1.2.0, django-cors-headers
- 前端：Vue 2
- 运行环境：Python 3.7.7, MySQL 5.7, Node 12, cnpm 7.1.0

## 功能模块

### 教师管理

- 管理教师信息

### 学生管理

- 管理学生信息

### 公告信息管理

- 发布、更新及管理公告信息

### 课程类型管理

- 管理课程类型及分类

### 课程信息管理

- 管理课程详情及安排

### 选课信息管理

- 管理学生选课情况

### 成绩信息管理

- 录入、查询及分析学生成绩

## 系统角色

- 教师：负责录入和管理学生成绩，发布公告，管理课程信息。
- 学生：查看成绩，选择课程，关注公告。
- 管理员：维护系统运行，管理用户和权限。

## 运行环境

- Python 3.7.7
- MySQL 5.7
- PyCharm 3（可选开发工具）
- Node 12
- cnpm 7.1.0

## 核心亮点

- 全面模块化的功能设计，方便扩展与维护。
- 利用 django-threadlocals 处理多线程环境下的数据隔离，保障数据安全。
- 支持跨域资源共享，提升前后端分离架构下的数据交互灵活性。
